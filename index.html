    入庫報告書 PDF 取込

    ミズタから届いた入庫報告書 (.pdf) をアップロードしてください。
    商品バリエーションには商品コード／商品名／入数を追加（既存コードは何もしません）。
    在庫台帳には入庫数／ケース数／内バラ数を追加（既存コードは加算します）。
    アップロード後、内容を確認してから「反映」を押してください。

    弥生商品マスタの更新

    弥生からエクスポートした商品マスタ (.xlsx) をアップロードすると、その場で商品マスタを更新します。
    ※商品リストを一切加工せずにアップロードしてください。(加工するとバグります)

    在庫一覧

    登録されている商品の在庫数を一覧表示します。

    1. 左の検索欄にコード or 商品名を入れると絞り込み
    2. 分類4プルダウンでブランド絞り込み
    3. 「在庫0も表示」チェックで在庫切れ品も表示
    4. 右上「CSV」で表示中のリストをExcel取込用にダウンロード
    引き寄せ依頼

    ミズタ倉庫から自社へ商品を取り寄せるための依頼書（PDF）を作成します。
    確定すると当方在庫が減算されます。

    1. 依頼日・配送方法（トラック/佐川）・希望到着日を入力
    2. 「＋ 行追加」で品目を追加。検索ボックスからマスタを選んで数量入力
    3. 右側に在庫数が表示される。在庫超過は赤塗りで警告
    4. 「プレビュー」→ PDFを確認
    5. 「確定して下書き作成」→ Gmail下書きにPDFが添付＋在庫減算＋ログ追加
    6. 同一着日のケース合計が 80未満/100超のとき注意警告が出る
    返送依頼

    ミズタ倉庫へ商品を返送するための依頼書（Excel）を作成します。
    在庫台帳は触りません（在庫の動きは「入庫報告書取込」で別途反映）。

    ＋ 新規作成

    1. 入庫予定日を入力
    2. 「＋ 行追加」で品目行を追加し、各セルを直接入力（または検索ボックスからマスタ候補選択）
      • マスタに無いコードや入数違いも、その場で書き換えられる
      • ミズタ商品名は半角36バイト以内（文字数列が赤くなったらNG）
      • 入数が空 or 0だと赤くなる
    3. 「下書き保存」で途中保存（→ 同時にDriveにコピペ用スプシが生成される）
    4. 「プレビュー」を押すとPDFプレビューが表示される
    5. 内容OKなら「確定して Excel下書き作成」 → Gmail下書きに Excel が添付される

    下書き

    1. 過去に保存した返送下書きが一覧表示される
    2. 「スプシ」で下書きスプシを開く（段ボール貼付のコピペ用）
    3. 「開く」でフォームに展開、修正して保存 or 確定
    4. 「削除」で下書きとスプシをまとめて削除
    在庫照合（月次）

    ミズタから届いた月次の在庫報告書（Excel）と当方DBを突き合わせて差異を確認・反映します。
    未登録の新商品もここで一括登録できます。

    1. ミズタ在庫報告書 (.xlsx) を選んで「インポート・差異確認」
    2. 差異一覧が表示される（緑=多い／赤=少ない／ピンク=未登録）
    3. 未登録行は弥生コード/弥生名を入れて「新規商品を登録」
    4. 各差異の取扱い（Excelに合わせる/システム維持）を選んで「差異を確定」
    入庫報告書取込

    ミズタから届く入庫報告書PDFを取り込み、商品バリエーション（新規分）と在庫台帳を更新します。
    返送系で在庫が動かなくなった分、ここで一括反映する運用です。

    1. 入庫報告書 (.pdf) を選んで「解析してプレビュー」
    2. 新規追加/加算更新の内訳をプレビューで確認
    3. 「スプレッドシートに反映」で書き込み

    ※ バリエーションは新規だけ追加（既存コードはスキップ）／在庫台帳は既存なら加算更新

    依頼書履歴

    過去に作成した依頼書（引き寄せ/返送）の一覧と管理。

    1. 種別プルダウンで絞り込み
    2. 各行から Gmail / PDF / Excel を開ける
    3. 「受け取り済みにする」「返送済みにする」ボタンでステータス更新
    4. 左のチェックを入れて「選択削除」で削除（引き寄せは在庫を戻す、返送は在庫据え置き）
    マスタ更新

    弥生からエクスポートした商品マスタを丸ごと置換します。

    1. 弥生商品マスタ (.xlsx) を選択
    2. 「アップロードして更新」

    ※ 商品リストは加工せずにアップロード。加工するとバグる

    マスタDBを開く（右上ボタン）

    スプレッドシート本体を別タブで開きます。シート構成・依頼書ログ・下書き操作ログなどの裏側を直接確認したいときに使います。

    引き寄せ依頼 下書き一覧

    保存した下書きを開いて編集・再利用できます。共有なのでメンバー全員の下書きが見えます。

    返送依頼 下書き一覧

    保存した下書きを開いて編集・再利用できます。共有なのでメンバー全員の下書きが見えます。
